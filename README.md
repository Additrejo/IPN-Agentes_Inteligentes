# Sistema de Análisis de Sentimientos Basado en Léxico

##  Descripción

Sistema de clasificación de sentimientos que implementa un enfoque basado en reglas y diccionarios para analizar comentarios de usuarios en Facebook. El proyecto procesa 500 comentarios reales, identificando patrones lingüísticos y clasificando el sentimiento expresado en cada mensaje.

##  Aspectos técnicos:
- **Paradigma**: Sistema basado en reglas (no ML)
- **Técnica**: Análisis léxico con diccionarios
- **Procesamiento**: Tokenización, limpieza y normalización de texto
- **Output**: Reportes Excel con métricas detalladas
- **Interfaz**: GUI desarrollada con Tkinter

## Resultados:
- 500 comentarios analizados
- 6,556 palabras en diccionarios (positivas + negativas)
- Clasificación en tiempo real
- Visualización de top 10 palabras por categoría

## Valor educativo:
Este proyecto demuestra conceptos fundamentales de NLP y sistemas basados en conocimiento, sirviendo como base para futuras implementaciones con machine learning.
